DungeonDex v1.4.3d - Smoke Notes & Recovery Polish

Utility/recovery hotfix only. No combat balance, loot balance, dungeon progression, or save structure changes.

Changed:
- Updated visible version label to DungeonDex v1.4.3d.
- Updated cache/build slug to dungeon-dex-v1.4.3d-smoke-notes-recovery-polish.
- Added an Archive > Build Tools section with current version, cache/build slug, and recovery helper text.
- Added Clear Cache & Reload, Export Save, and Copy Build Info buttons.

Notes:
- Clear Cache & Reload unregisters service workers and clears browser caches where supported, then reloads. It does not touch save data.
- Export Save reads the current localStorage save payload and downloads it as JSON, with a copy/manual fallback when downloads are unavailable.
- Copy Build Info copies a plain text diagnostic report, with a manual textarea fallback when clipboard access is unavailable.
